# AI Product Manager for Founders and Developer Teams

An AI product manager is a workflow for turning raw ideas into clearer strategy, roadmap, and execution tasks. It is especially useful for founders and small teams building AI coding tools, developer utilities, SaaS products, and agent-powered workflows.

## What an AI Product Manager Helps With

- **Search intent mapping**: Convert keywords and user problems into page, product, or feature ideas.
- **Feature prioritization**: Decide what should be MVP, P1, P2, or not now.
- **PRD drafting**: Create implementation-ready requirements for developers or coding agents.
- **Roadmap planning**: Group feature work by audience, revenue path, and technical dependency.
- **Acceptance criteria**: Define what must be true before a feature is considered shipped.

## When to Use It

Use an AI product manager when you have a promising product idea but need a sharper plan before writing code. It can help with early-stage SaaS tools, landing pages, internal automations, marketplace pages, and product-led SEO pages.

AI coding tools are strongest when given a clear product target. The AI product manager step defines the right feature and acceptance criteria; the AI PRD generator turns it into a spec; coding tools then implement and test the scoped work.
